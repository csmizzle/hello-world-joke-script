A simple Python script that fetches random jokes from an API and displays them in your terminal.

## Features

- Fetches jokes from the [Official Joke API](https://official-joke-api.appspot.com/)
- Clean, formatted output
- Error handling for network issues
- Easy to use and extend

## Requirements

- Python 3.x
- `requests` library

## Installation

```bash
pip install requests
```

## Usage

Simply run the script:

```bash
python3 joke_fetcher.py
```

Or make it executable and run directly:

```bash
chmod +x joke_fetcher.py
./joke_fetcher.py
```
